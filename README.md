# KAIN - Knowledge Augmented Intelligent Network

KAIN is a professional coding assistant built with a modern full-stack architecture.

## Tech Stack
- **Frontend**: React + Vite
- **Backend**: FastAPI (Python)
- **AI**: Gemini 1.5 Flash
- **Knowledge**: RAG with local vector storage
- **Security**: JWT Authentication

## How to Run

### Backend
1. Go to `backend` directory.
2. Install dependencies: `pip install -r requirements.txt`
3. Set up `.env` with your `GEMINI_API_KEY`.
4. Run: `python main.py`

### Frontend
1. Go to `frontend` directory.
2. Install dependencies: `npm install`
3. Run: `npm run dev`

## Features
- Professional & Helpful tone.
- Hallucination-free answers using RAG.
- Code syntax highlighting.
- Responsive dark mode UI.
